# App Icons

Place app icons in this directory.

## macOS Icon Dimensions

For an authentic macOS look, use these standard icon sizes:

### Recommended Sizes:
- **256x256px** - Primary size for dock and desktop icons (best for web)
- **512x512px** - Retina display quality
- **1024x1024px** - Source file quality (for design assets)

### Standard macOS Icon Sizes:
- 16x16px - Very small (menu bar)
- 32x32px - Small icons
- 64x64px - Medium icons
- 128x128px - Large icons
- 256x256px - Dock/Desktop (1x)
- 512x512px - Retina @2x
- 1024x1024px - Retina @3x

## File Naming Convention:
Use descriptive names matching the app ID:
- `artifact-explorer.png`
- `job-monitor.png`
- `agent-console.png`
- `settings.png`

## Format:
- **PNG** with transparency (recommended)
- **SVG** for scalable vector icons (also great for web)

## Design Guidelines:
For a macOS-like appearance:
- Use rounded square backgrounds (iOS/macOS style)
- Apply subtle gradients
- Add soft shadows for depth
- Use the macOS icon grid system
- Border radius: ~22-24% of icon size for that authentic macOS look
